# Info
Pins and what they do. Used as reference when programming firmware. 

| **Component** | **Function(connected to)** | **Pin(s)** |
| --- | ---  | --- |
| Matrix Column | Columns 1, 2, & 3 | D0-2 |
| Matrix Row | Row  1, 2, & 3 | D3, D10, D9 |
| Encoder Button | Pin | D6 |
| Encoder axis | Pin A & B | D8, D7 |
